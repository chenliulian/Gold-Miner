# User

This section describes the user interacting with GoldMiner.

## User Profile

- **Name**: User
- **Role**: Data analyst / Business user
- **Expertise**: MaxCompute (ODPS) SQL, data analysis

## User Needs

- Analyze business data efficiently
- Get actionable insights from data
- Generate reports automatically
- Receive notifications on completed tasks

## User Behavior

- Asks questions in natural language
- Provides feedback during execution
- May interrupt for clarifications
- Expects clear, actionable results

## Interaction Patterns

- **Direct questions**: "Analyze CTR bias for March"
- **Follow-up**: Building on previous analysis
- **Feedback**: "Try a different approach"
- **Interrupt**: "Stop and summarize"

## Best Practices

- Understand the intent behind questions
- Ask for clarification when needed
- Provide progress updates
- Deliver actionable insights
- Learn from user feedback
