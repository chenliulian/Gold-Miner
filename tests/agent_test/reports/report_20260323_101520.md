如需深入分析，可以：

2. 分国家/代码位类型下钻
3. 对比历史同期数据
4. 分析 CTR/CVR 模型预估偏差 (PCOC)

```sql
SELECT 
    SUBSTR(dh, 1, 8) as dt,
    ad_group_id,
    SUM(show_cnt) as show_cnt,
    SUM(click_cnt) as click_cnt,
    SUM(pv) as conv_cnt,
    ROUND(SUM(click_cnt) * 100.0 / NULLIF(SUM(show_cnt), 0), 4) as ctr,
    ROUND(SUM(pv) * 100.0 / NULLIF(SUM(click_cnt), 0), 4) as cvr,
    ROUND((SUM(cpm_cost) + SUM(cpc_cost)), 2) as cost_usd
FROM com_cdm.dws_tracker_ad_cpc_cost_hi
WHERE dh >= SUBSTR(REPLACE(CAST(DATEADD(TO_DATE(MAX_PT('com_cdm.dws_tracker_ad_cpc_cost_hi'), 'YYYYMMDDHH'), -7, 'HH') AS STRING), '-', ''), 1, 10)
AND advertiser_id = '2368'
GROUP BY SUBSTR(dh, 1, 8), ad_group_id
ORDER BY dt DESC, ad_group_id
LIMIT 1000
```
