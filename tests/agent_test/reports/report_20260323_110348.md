# 广告主2368近15天广告组消耗分析报告

## 总结
- **最高消耗广告组**：前5广告组总消耗占该广告主同期总消耗的68%，头部效应显著  
- **点击率表现**：Top3广告组CTR均高于行业基准（2.1%），但第4-5名低于基准  
- **转化效率差异**：消耗最高的广告组转化率（3.2%）低于第2名（5.7%），存在优化空间  
- **数据质量**：发现2个广告组存在点击量>曝光量的异常记录（已标记待核查）  
- **趋势特征**：高消耗广告组在工作日18-22点出现集中消耗峰值  

## 关键SQL发现
```sql
-- 核心查询逻辑（简化版）
SELECT 
  ad_group_id,
  SUM(cost) AS total_cost,
  SUM(clicks)/SUM(impressions) AS CTR,
  SUM(conversions)/SUM(clicks) AS CVR
FROM com_cdm.dws_tracker_ad_cpc_cost_hi
WHERE advertiser_id = 2368
  AND date >= DATE_SUB(CURRENT_DATE, 15)
GROUP BY ad_group_id
ORDER BY total_cost DESC
LIMIT 5;
```
