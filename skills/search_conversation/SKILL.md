# 对话历史检索

## 名称
search_conversation

## 描述
检索历史对话记录，支持按关键词搜索或查看最近的对话。

## 参数
- query: 搜索关键词 (可选)
- limit: 返回数量 (可选，默认 5)

## 使用场景

### 1. 搜索包含关键词的历史
```
query: "CTR analysis"
```

### 2. 查看最近的对话
```
query: ""
limit: 10
```

## 输出
返回匹配的对话历史列表，包含时间、问题、简要回答。

## 注意事项
- 搜索memory目录下的历史记录
- 支持模糊匹配
- 按时间倒序排列
