# Heartbeat

## 名称
heartbeat

## 描述
Agent 心跳机制，定期发送进度更新。用于长时间运行的任务中保持连接，报告进度。

## 参数
- message: 心跳消息
- progress: 进度百分比 (可选)

## 使用场景

### 1. 报告进度
```json
{
  "message": "已完成数据查询",
  "progress": 50
}
```

### 2. 保持活跃
```json
{
  "message": "任务进行中..."
}
```

## 注意事项
- 每 3 个步骤自动触发一次心跳
- 用于向用户报告执行进度
- 不影响任务执行
