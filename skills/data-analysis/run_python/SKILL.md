# run_python

## 名称
run_python

## 描述
编写并执行Python脚本进行数据分析。支持pandas/numpy等数据分析库，可以对SQL查询结果进行二次加工、统计分析、数据清洗等操作。执行脚本后返回stdout输出、变量结果和错误信息。适用于：1) SQL难以实现的复杂计算 2) 数据可视化准备 3) 统计检验 4) 自定义数据转换

## 参数
- dataframe: 输入数据 (可选)

## 输出
返回处理结果

- 编写并执行Python脚本进行数据分析。支持pandas/numpy等数据分析库，可以对SQL查询结果进行二次加工、统计分析、数据清洗等操作。执行脚本后返回stdout输出、变量结果和错误信息。适用于：1) SQL难以实现的复杂计算 2) 数据可视化准备 3) 统计检验 4) 自定义数据转换
